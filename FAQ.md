## 💻 Hardware & Performance / Hardware y Rendimiento

### [EN] Is my GPU supported?
Gravity supports **NVIDIA CUDA** and **AMD ROCm/DirectML**. If you have an integrated GPU (Radeon 780M+), you can use it too.

### [ES] ¿Mi GPU es compatible?
Gravity soporta **NVIDIA CUDA** y **AMD ROCm/DirectML**. Si tienes una GPU integrada (Radeon 780M+), también puedes usarla.

---

## 🛡️ Privacy & Security / Privacidad y Seguridad

### [EN] Is my data sent to the cloud?
By default, **NO**. Gravity prioritizes local hardware. Only if you configure a cloud key in `KeyManager` will requests be sent to external providers.

### [ES] ¿Mis datos se envían a la nube?
Por defecto, **NO**. Gravity prioriza el hardware local. Solo si configuras una llave de nube en el `KeyManager`, las peticiones se enviarán a proveedores externos.
