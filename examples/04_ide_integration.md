# Integración con Cursor, Aider y Continue.dev

Este documento proporciona ejemplos y una guía de configuración rápida para integrar las herramientas Cursor, Aider y Continue.dev utilizando la API local del bridge de Gravity AI.

## Requisitos Previos
- Tener instalado el software del bridge de Gravity AI.
- Contar con acceso a la API local.

## Integración con Cursor
1. **Instalar Cursor:**
   - Descarga e instala la aplicación Cursor desde [su sitio oficial](https://cursor.so).
2. **Configurar API:**
   - Abre la configuración de Cursor y agrega la URL de la API local del bridge.
3. **Ejemplo de Uso:**
   ```json
   {
       "query": "Tu consulta aquí"
   }
   ```

## Integración con Aider
1. **Instalar Aider:**
   - Descarga e instala la aplicación Aider desde [su sitio oficial](https://aider.ai).
2. **Configurar API:**
   - Abre la configuración de Aider y agrega la URL de la API local del bridge.
3. **Ejemplo de Uso:**
   ```json
   {
       "input": "Datos requeridos"
   }
   ```

## Integración con Continue.dev
1. **Instalar Continue.dev:**
   - Accede a Continue.dev y regístrate en su plataforma.
2. **Configurar API:**
   - Dentro de Continue.dev, agrega la URL de la API local del bridge.
3. **Ejemplo de Uso:**
   ```json
   {
       "parameters": {
           "param1": "valor1",
           "param2": "valor2"
       }
   }
   ```

## Conclusión
Siguiendo estos pasos, podrás integrar efectivamente el bridge de Gravity AI con Cursor, Aider y Continue.dev usando su API local. No dudes en consultar la documentación específica de cada herramienta para obtener más detalles.
